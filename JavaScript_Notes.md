# DATA TYPES

JS has 7 data types.

- Undefine: Hasn't been defined, a variable that hasn't been set to anything
- Null: Nothing
- Boolean: True or False
- String: A string
- Symbol: An immutable primitive value that is unique
- Number: A number
- Object: Can store key-value pairs
